# Professional Experience & Portfolio Overview

My experience is best represented by complex full-stack applications, scalable AWS cloud infrastructures, and secure financial automation systems. 

While most of my work is under NDA or proprietary, the projects that best highlight my capabilities include:

### Enterprise IoT & Cloud Systems (Sigaway)
For a corporate fleet tracking company, I built and maintained scalable architectures using AWS (Kinesis Firehose, Lambda, S3, Athena, and DynamoDB). I worked across the entire stack, from integrating hardware telemetry via Python to developing backend modules (Java/Spring Boot, Node.js) and redesigning the React/Redux frontend. I also configured and managed CI/CD pipelines for automated deployments.

### Secure Financial Automation Platforms (ElliotFinance)
I engineered a complete investment ecosystem featuring an optimized Python trading bot integrated with the Binance exchange, alongside a highly secure Java/Spring Boot backend and PostgreSQL. The system runs on a scalable AWS infrastructure with a React/TypeScript frontend for real-time portfolio management, dynamic bot configuration, and external exchange API integrations. To guarantee high performance and strict security with sensitive encrypted data, the architecture separates the application and processing core instances, and user creation is restricted entirely to Admin accounts.

### Business Management SaaS & Startups
I have developed complete products from the ground up, such as:
* **HSAgenda:** A salon management system built with React, TypeScript, Material UI, and a Node.js/MongoDB backend. It operated successfully for 6 months and received a startup incubation offer.
* **Firmament Inovação Corporativa:** An HR team management platform built with Node.js and Prisma ORM.
* **CPOL:** Customizable CMS-style portals for university research centers using modern React, Redux, and Supabase architectures.
